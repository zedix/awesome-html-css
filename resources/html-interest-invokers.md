# Interest Invokers API (`[interesttarget]`)

```html
<button interesttarget="profile-callout">
  ...
</button>

<div id="profile-callout" popover>
  ...
</div>
```

```css
[interesttarget] {
  /* Entry Exit delay */
  interest-target-delay: 0s 1s;

  &:has-interest {
    background-color: yellow;
  }
}
```
